## Parch and Posey Analysis by Pipeline_Pioneers (Circle 4)
In this repository, we explored the Parch and Posey Database to derive insights from their business.
The database contains 5 csv data models:
- web_events.csv
- sales_reps.csv
- region.csv
- orders.csv
- accounts.csv

We answered several questions to explore the factors that can boost profitability and streamline operations. These have been reported in our PowerPoint presentation. (see link here: [canva](https://www.canva.com/design/DAGyb07pg8I/S90OF9PMRQpw_WZ0Rlu1Lg/view) )
